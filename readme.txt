Add Bank Saman payment gateway to easy digital downloads transactions

== Description ==
This plugin will add Bank Saman gateway to easy digital downloads plugin

Features:

* 100% Free.
* Add Bank Samanand Shaparak Electronic Network gateway for selling files.
* Change currency "RIAL" to "ریال" in persian language.
* Add "Toman" currency.
* Without any advertising.
* Support Test Mode

Language Support:

* Persian
* English
* Arabic

NOTE : before install this plugin, you should installed [Easy Digital Downloads](https://wordpress.org/plugins/easy-digital-downloads/ "Easy Digital Downloads")

Have any problem? [Get in touch](https://arvandec.com/contact "Arvandec Contact Us")

== Installation ==
1. Install and active "EasyDigitalDownloads".
2. Upload 'bank-saman-edd-gateway' to the '/wp-content/plugins/' directory.
3. Activate the plugin through the "Plugins" menu in WordPress.
4. Go to '/wp-admin/edit.php?post_type=download&page=edd-settings&tab=gateways' and active Bank Saman
